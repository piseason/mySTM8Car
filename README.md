## 物料
电机是立华电子原价3.6元的那一款，电池也是立华电子的，原价1.9元。飞线用的是0.35mm的漆包线，其它的都是通用模块，直接在淘宝搜就行，都差不多。参考价：STM8核心板3块9，万能板4毛，2mm钻头3块，连轴器1块5，14500电池4块，MPU6050(3块2)，N20电机支架3毛5，DRV8833(2块4)，车轮1块2
